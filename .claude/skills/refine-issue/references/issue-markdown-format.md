# Issue Markdown Format

```markdown
---
number: 123
title: タスク一覧画面に優先度カラムを追加
url: https://github.com/OWNER/REPO/issues/123
---
## 背景・目的
...

## 要件
...
```

## ルール

- `number` は変更しない
- `title` を編集すると Issue タイトルも更新される
- frontmatter の後ろ全体が Issue 本文として GitHub に反映される
- 本文は GitHub Flavored Markdown として扱う
